# CropScan — Frontend

Live at: **https://robisharp.github.io/cropscan_api/**

Mobile-friendly crop disease detection interface.  
Calls the [CropScan API](https://github.com/robisharp/cropscan-api) backend hosted on Render.

## Stack
- Vanilla HTML/CSS/JS — zero dependencies, no build step
- Hosted on GitHub Pages

## Local dev
Just open `index.html` in your browser. To point at a local backend, change `API_BASE` in the script section to `http://localhost:8000`.
